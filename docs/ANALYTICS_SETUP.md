# Analytics & Search Setup Guide

Last updated: 2026-04-26

## Purpose

Install measurement tools so traffic and conversions can be reviewed.

## Tools

- Google Analytics 4 (GA4)
- Google Search Console

## GA4 Steps

1. Create a GA4 property.
2. Copy the site tag.
3. Add the tag to the website `<head>`.
4. Publish changes.
5. Confirm page views are being received.

## Search Console Steps

1. Add your domain property.
2. Verify ownership.
3. Submit sitemap.
4. Review indexing status.
5. Monitor search queries over time.

## What To Track

- Total visits
- Traffic sources
- Top pages
- CTA clicks
- Form submissions
- Calls or inquiry actions
- Search impressions
- Search clicks

## Rule

Do not guess what is working when measurement can answer it.

## After Setup

Update `docs/ROADMAP.md` and check off the analytics tasks.
