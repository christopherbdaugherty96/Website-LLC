# Client Onboarding Checklist

## Before Build Starts
- Confirm business name
- Confirm phone / email / address
- Confirm services offered
- Collect logo / photos
- Collect preferred colors or style examples
- Confirm package selected
- Confirm payment / deposit

## During Build
- Send first version
- Collect feedback
- Apply revisions
- Confirm final approval

## Launch
- Publish website
- Test mobile
- Test buttons/forms
- Share live link

## After Launch
- Request testimonial
- Ask for referrals
- Offer future updates
