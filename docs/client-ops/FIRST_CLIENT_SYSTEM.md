# First Client System

## Purpose
Turn interest into the first paying website client using a simple repeatable process.

## Pipeline Stages
1. Find Prospect
2. First Contact
3. Follow Up
4. Discovery Conversation
5. Proposal Sent
6. Closed Won
7. Build & Launch
8. Testimonial / Referral

## Daily Minimum Standard
- 10 prospects researched
- 5 contacts sent
- 2 follow-ups
- 1 active conversation moved forward

## Core Rule
Revenue comes from consistent conversations, not endless redesign.
