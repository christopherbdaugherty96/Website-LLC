# Hosting and Domain Checklist

Status: operating checklist

Use this before launch, during migrations, or when taking over an existing client website.

---

## 1. Ownership Check

- Who currently owns the domain?
- Where is the domain registered?
- Who has login access?
- Who receives renewal emails?
- Who owns hosting?
- Who pays hosting/domain invoices?
- Is billing active?
- Is two-factor authentication enabled?

## 2. Access Check

- Registrar login available
- Hosting login available
- DNS access available
- Website platform admin access available
- Google Business Profile access (if relevant)
- Google Analytics / Search Console access (if relevant)
- Email provider access (if relevant)

## 3. Domain Health

- Domain not expired
- Auto-renew enabled
- Correct contact email on file
- Nameservers known
- DNS records reviewed
- SSL active
- Redirects working (www/non-www)

## 4. Hosting Health

- Hosting active
- Storage/bandwidth sufficient
- Backups enabled
- SSL active
- Performance acceptable
- Error logs reviewed
- Contact form working
- Current software up to date

## 5. Migration Safety

- Full backup created
- Existing site archived
- Rollback plan documented
- DNS TTL checked
- Launch window chosen
- Test environment ready
- Mobile QA completed
- Links/forms tested

## 6. New Build Setup

- Repo connected (GitHub/Vercel/Netlify)
- Environment variables set (if needed)
- Build settings confirmed
- Custom domain connected
- HTTPS active
- Analytics connected
- Search Console submitted
- Sitemap available

## 7. Payment Responsibility

Default recommendation:

- Client pays for domain renewals
- Client pays for hosting unless included in a written care plan
- Auralis may assist setup and management
- Any Auralis-managed billing must be documented clearly

## 8. Final Launch Signoff

- Client approved final website
- Contact details confirmed
- Hours confirmed
- Services/pricing confirmed
- Leads route correctly
- Ownership confirmed
- Access credentials stored safely
- Support plan chosen (none / one-time / monthly care plan)
