# Public Launch Checklist

Last updated: 2026-04-26

Use this checklist before actively sending prospects to the website.

## Brand / Trust

- [ ] Public pages use Auralis Digital branding
- [ ] No visible Y.O.U. / Your Own Vision references remain where they should be updated
- [ ] Founder/about page reflects current business identity
- [ ] Contact details are consistent across public pages

## Lead Capture

- [ ] Contact form works end-to-end
- [ ] Email fallback still works
- [ ] Inquiry path is clear on mobile and desktop
- [ ] CTA buttons are visible and understandable

## Content Quality

- [ ] Demo pages shared with prospects have no placeholders
- [ ] No fake testimonials are shown
- [ ] Pricing language is clear
- [ ] Service descriptions are understandable

## Technical Checks

- [ ] Homepage loads correctly on mobile
- [ ] Main navigation works
- [ ] Key links work
- [ ] Sitemap exists and is current
- [ ] Robots file exists and points to sitemap

## Measurement

- [ ] Google Analytics 4 installed
- [ ] Search Console connected
- [ ] Traffic and conversions can be reviewed

## Sales Readiness

- [ ] Lead tracker ready
- [ ] Outreach messages ready
- [ ] Proposal template ready
- [ ] Onboarding flow ready for first client

## Rule

Do not wait for perfection. Fix credibility issues first, then start outreach.
